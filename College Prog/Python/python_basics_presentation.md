# Python Basics Presentation


## Introduction
Python is a high-level, interpreted programming language known for its simplicity and readability. It was created by Guido Van Rossum in 1991. It is widely used in web development, data science, automation, and more.

## Why Learn Python?
- Easy to read and write
- Large community support
- Extensive libraries and frameworks
- Versatile (web, data, automation, AI)

## Basic Syntax
```python
print("Hello, World!")
```

## Variables and Data Types
```python
x = 5           # Integer
name = "Alice"  # String
pi = 3.14       # Float
is_valid = True # Boolean
```

## Input and Output
```python
name = input("Enter your name: ")
print("Hello,", name)
```




## For Loop

**Syntax:**
```python
for variable in sequence:
    # code block
```

**Example:**
```python
for i in range(5):
    print(i)
```

## Conclusion
Python is beginner-friendly and powerful. Start practicing to master the basics!
